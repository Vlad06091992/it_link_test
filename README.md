## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:app.e2e
```

## Usage

```bash
local url: http://localhost:3000/graphql

Данные для того, чтобы наполнить БД, и примеры GraphQL запросов можно найти в папке /dev.
```
