## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Usage

```bash
local url: http://localhost:3000/graphql

Данные для того, чтобы наполнить БД, и примеры GraphQL запросов можно найти в папке /dev.
```
